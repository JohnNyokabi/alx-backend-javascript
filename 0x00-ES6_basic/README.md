# 0x00. ES6 Basics

## Requirements
        * Ubuntu 20.04LTS
        * vi/vim/emacs/VSCode
        * Jest Testing Framework
        * NodeJS 12.11.x and above

## Installations
        $ npm install --save-dev jest
        $ npm install --save-dev babel-jest @babel/core @babel/preset-env
        $ npm install --save-dev eslint
        $ npm install
